About Pneumonia Detection

This project is a real-time pneumonia detection system that uses deep learning on chest X-ray images. The model predicts whether the X-ray shows a healthy lung or signs of pneumonia.

Features include:

  • Accurate detection using a CNN-based model (VGG16 or similar)
  • Interactive web interface for uploading X-ray images
  • Model evaluation metrics and confusion matrix visualization
  • Clinical recommendations based on results

This tool is intended to assist medical professionals and is not a substitute for professional medical advice.
